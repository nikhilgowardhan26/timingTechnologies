The Certificates are mostly made available for download after a week from the date of publishing the event results onto the website. All the certificates will be in PDF format.

The results are sent to the event organiser on the same day of the event date. The event organiser sends a confirmation back to Timing Technologies to publish the results. Upon confirmation, the results are made live on to the Timing Technologies website very immediately.

Your details reflecting on the results page are based on the data provided to us by the event organiser. For any changes in this data, please contact the event organiser.

You need to write a mail to the event organiser where their mail id is made available on the results page of the Timing Technologies website. Once we get an approval from the event organiser, we take action to make necessary changes.

NET Time (CHIP Time) is the individual start time to the individual finish time.

GROSS Time (GUN Time) is the time from gun start of the race to the individual finish time.

GUN start is the official start time of the run.
